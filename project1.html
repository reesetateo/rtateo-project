This personal project involved the cleaning and transformation of publicly accessible tourism datasets to create an interactive dashboard depicting the effects of the COVID-19 pandemic on commodity consumption.

Utilizing BigQuery for data processing and Looker Studio for visualization, I developed a dashboard that allows users to select and compare various commodities, offering insights into how consumption patterns have shifted due to the pandemic. For further exploration and to interact with the dashboard, please see the embedded report above. Users can select desired commodities through the dropdown filter to view data through the chart and table.
