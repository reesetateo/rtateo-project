QuoteQuest is a dynamic AI model designed to deliver personalized and meaningful quotes based on user input.

Visit QuoteQuest

The project utilizes GPT-4o Mini to generate relevant insights from a curated dataset inspired by my dad's endless cycle of reading and supplying me with daily advice, which has been compiled into a set of quotes that I have personally learned from throughout my life. Developed using React for the frontend and Flask for the backend, this project connects seamlessly to an AI model that analyzes user prompts and retrieves the most fitting quotes from the database.
